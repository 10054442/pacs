Write a different program for each of the following point.
\begin{itemize}

    \item Statically check if a template parameter of a class is \cpp{int} or
    \cpp{double}; rise an error otherwise.

    \item Declare a function that print a template variable. Statically check
    if the latter is not a raw pointer.

    \item Declare \cpp{unsigned int i}, then declear a \cpp{float} or an \cpp{int}
    if \cpp{i} is signed or unsigned without explicity know the type of \cpp{i}.
    Check the result.

\end{itemize}
